关于禁止航空运输三星Galaxy Note7手机的规定

鉴于三星Galaxy Note 手机存在异常发热、燃烧等问题，有可能引发起火，为确保航空运输安全，根据民航局10月25日下发的《关于禁止航空运输三星Galaxy Note 7手机的公告》，自10月27日起广西北部湾航空禁止旅客和机组成员携带三星Galaxy Note 7手机乘机，禁止将三星Galaxy Note 7手机作为航空货物运输，请旅客朋友们注意：

一、旅客不得随身或在手提行李中携带三星Galaxy Note 7 手机乘机。

二、旅客不得将三星Galaxy Note 7手机放入托运行李中托运。

三、严禁把三星Galaxy Note 7手机作为航空货物收运。

四、按照中国民航局《关于严禁航空运输三星Galaxy Note 7手机的公告》规定，北部湾航空将加大巡查、检查力度，及时发现并制止携带Galaxy Note7手机或其锂电池乘机或托运的行为。

五、对于违反上述规定者，公安机关、民航行政机关将根据情节，依照国家有关法律、法规严肃处理。

特此通知，感谢您的关注与支持。
